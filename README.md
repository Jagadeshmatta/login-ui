# Smart Registration & Login System

A responsive and user-friendly Registration and Login Page built using HTML, CSS, and JavaScript. This project provides a clean authentication interface with modern UI design and basic form validation.

---

## 🚀 Features

- User Registration Form
- User Login Form
- Responsive Design
- Form Validation using JavaScript
- Clean and Modern UI
- Easy Navigation Between Pages
- Beginner-Friendly Project Structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
